# API Films – PHP

API REST simple en PHP permettant de récupérer des films depuis l’API The Movie Database (TMDB) et de renvoyer les données au format JSON.

---

## Installation

### Prérequis
- PHP installé sur la machine
- Une clé API TMDB

### Configuration
Créer un fichier `config/config.php` à partir de `config.example.php` et y ajouter la clé API TMDB.

### Lancer le serveur
Depuis le dossier du projet :

```bash
php -S localhost:8000
